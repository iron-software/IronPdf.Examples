# How to Merge or Split PDFs

***Based on <https://ironpdf.com/how-to/merge-or-split-pdfs/>***


Combining several PDF files into one streamlines sharing and managing documents, such as merging all corresponding resumes into a unified document. This tutorial demonstrates how to use C# and IronPDF for merging and splitting PDF files, offering easy-to-follow steps and methods appropriate for any C# development environment.

## Example of Merging PDFs

In this example, we will create two PDFs from HTML content and then merge these into a single document using IronPDF. Here's how it's done:

```cs
using IronPdf;
namespace ironpdf.MergeOrSplitPdfs
{
    public class MergeSection
    {
        public void Execute()
        {
            // HTML content for the first two-page PDF
            const string htmlPart1 =
                @"<p>Content of PDF 1</p>
                <p>Page 1 of PDF 1</p>
                <div style='page-break-after: always;'></div>
                <p>Page 2 of PDF 1</p>";
            
            // HTML content for the second two-page PDF
            const string htmlPart2 =
                @"<p>Content of PDF 2</p>
                <p>Page 1 of PDF 2</p>
                <div style='page-break-after: always;'></div>
                <p>Page 2 of PDF 2</p>";
            
            var pdfRenderer = new ChromePdfRenderer();
            
            var pdf1 = pdfRenderer.RenderHtmlAsPdf(htmlPart1);
            var pdf2 = pdfRenderer.RenderHtmlAsPdf(htmlPart2);
            
            // Merging PDFs into a four-page document
            var mergedPdf = PdfDocument.Merge(pdf1, pdf2);
            mergedPdf.SaveAs("Merged.pdf");
        }
    }
}
```

## Combining PDF Pages

The `CombinePages` method enables combining multiple pages into one page by specifying layout dimensions and the grid.

```cs
using IronPdf;
namespace ironpdf.MergeOrSplitPdfs
{
    public class CombinePagesSection
    {
        public void Execute()
        {
            PdfDocument pdf = PdfDocument.FromFile("Merged.pdf");
            
            // Specifying dimensions in millimeters for the combined page
            PdfDocument combinedDocument = pdf.CombinePages(250, 250, 2, 2);
            
            combinedDocument.SaveAs("combinePages.pdf");
        }
    }
}
```

## Example of Splitting a PDF

This section shows how to separate a multi-page document into different PDFs.

```cs
using IronPdf;
namespace ironpdf.MergeOrSplitPdfs
{
    public class SplitSection
    {
        public void Execute()
        {
            // Starting with the merged 4-page PDF
            var pdf = PdfDocument.FromFile("Merged.pdf");
            
            // Separating the first page into a new PDF
            var firstPagePdf = pdf.CopyPage(0);
            firstPagePdf.SaveAs("Page1Only.pdf");
            
            // Extracting the second and third pages
            var middlePagesPdf = pdf.CopyPages(1, 2);
            middlePagesPdf.SaveAs("Pages2to3.pdf");
        }
    }
}
```
